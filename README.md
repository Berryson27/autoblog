# Naver Blog Writer

Electron + Playwright 기반 네이버 블로그 글쓰기 로컬 입력 앱입니다.

## 실행

1. `npm install`
2. `npm start`

## 현재 범위

- Playwright 브라우저 실행
- 사용자가 직접 네이버 로그인
- 블로그 글쓰기 화면 이동
- 제목, 텍스트 블록, 이미지 블록, 이미지 캡션, 태그 입력
- 입력 블록 순서 조정
- 네이버에 들어갈 순서 미리보기
- 로컬 이미지 파일 메타데이터 구성
- 초안을 네이버 글쓰기 화면에 순서대로 입력
- 발행은 자동으로 누르지 않음

## 아직 하지 않는 부분

- 자동 발행
- CAPTCHA/2FA 우회
- Firebase/서버 저장
- AI 글 생성
