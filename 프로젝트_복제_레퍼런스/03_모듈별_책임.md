# 모듈별 책임

## package.json

- 앱 entry는 `src/main/main.js`
- 실행은 Electron
- 테스트는 Node test runner
- 주요 의존성은 Electron, Playwright, pngjs

## src/main/main.js

- Electron BrowserWindow 생성
- preload 연결
- main UI 창 크기 계산
- Playwright 브라우저 위치/크기 계산
- IPC handler 등록
- 파일 선택 dialog 처리
- clipboard 쓰기 처리
- Naver service와 Generation service 생성

## src/main/preload.js

- Renderer에 `window.autoclick` API 제공
- IPC invoke 래핑
- File 객체에서 로컬 path 추출
- 자동화 상태 이벤트 구독 제공

## src/main/services/browserSession.js

- Playwright persistent context 관리
- Chrome channel 우선 사용
- 실패 시 bundled Chromium fallback
- 로그인 세션을 userDataDir에 유지
- page console/error/dialog/crash 로그 연결

## src/main/services/naverBlogService.js

- 로그인 페이지 열기
- 블로그 글쓰기 페이지 열기
- payload 검증 후 draft output 생성
- 현재 page가 글쓰기 화면인지 보장
- editor service에 실제 입력 위임

## src/main/services/naverDraftPayload.js

- blogId로 글쓰기 URL 생성
- tag 문자열 정규화
- Renderer blocks를 Naver 입력용 blocks로 정규화
- 지원하지 않는 이미지 확장자 제거
- 빈 블록 제거
- quotation type 정규화
- payload 필수값 검증

## src/main/services/generationService.js

- Gemini API 호출 담당
- generation job Map 관리
- 이미지 개수 제한
- 이미지 파일 base64 변환
- prompt 생성
- API key 해석
- 과금/크레딧 관련 오류를 사용자 메시지로 변환

## src/main/services/imageAssetService.js

- 이미지 확장자별 MIME 판별
- 지원 이미지 여부 검사
- 파일 존재와 크기 확인
- 중복 경로 제거
- Renderer에서 쓰기 좋은 asset 형태로 반환

## src/main/services/naverEditorService.js

- 네이버 SmartEditor iframe 탐색
- 임시글 복구 팝업 닫기
- 제목 영역 포커스 및 입력
- 본문 영역 포커스
- 텍스트, 이미지, 소제목, 인용구, 구분선 삽입
- toolbar 스타일 조작
- 이미지 삽입 성공 확인
- editor-debug.log 기록
- frame detach 재시도
- 각 단계 timeout watchdog 적용

## src/renderer/index.html

- 앱 화면 구조
- 단계형 workflow UI
- 이미지 업로드 영역
- 기획값 입력 영역
- 생성 결과 편집 영역
- 네이버 제출/복사 버튼

## src/renderer/renderer.js

- 화면 이벤트 바인딩
- Renderer 상태 관리
- 이미지 추가/삭제/정렬/캡션 수정
- 생성 요청/결과 조회
- 버튼 활성화 조건 계산
- status log 출력
- 최종 Naver payload 제출

## src/renderer/draftModel.js

- Renderer 상태를 workflow draft로 정규화
- AI 응답에서 제목/본문/태그 분리
- 본문 마커를 block 배열로 변환
- 최종 Naver payload 생성
- 복사용 plain text 생성

## src/renderer/previewRenderer.js

- 최종 block 배열을 미리보기 형태로 렌더링
- 이미지, 텍스트, 태그 블록 구분 표시

## src/renderer/filePathView.js

- 로컬 path에서 파일명 추출
- 로컬 이미지 preview용 file URL 생성

## src/renderer/statusLog.js

- 최신 상태 메시지와 로그 목록 관리
- 로그 표시/숨김 토글

## tests

- draft model 변환 테스트
- file path view 테스트
- Naver payload 정규화 테스트
- image asset 테스트
- generation service job 및 prompt 테스트
